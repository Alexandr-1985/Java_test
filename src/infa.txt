1. За основу берем PDF файл с прорисованной схемой нашего проекта, на схеме указаны конечные js файлы, они должны писаться
первыми, дальше прорисовываются выше стоящие по цепочке
2. В созданный проект добавляем нехватающие нам библиотеки для работы с ним:
npm i bootstrap
npm i lodash
npm i font-awesome
npm i react-router-dom
3. Далее создаем коталог config (конфигурации) и внутри этой катигории создаем courses_configuration.js и добавляем в него constans
И только после мы можем построить CoursNav, тк он является каталогом нашего js файла.
return в  coursesNav - это реторн который вызывается из нашей функциональности и он будт вызываться для каждого элемента преобразующего из входного в выходное (ul)
4. Создаем дирикторию components. И строим компоненты навигатора, тк те файлы от которых наш навигатор уже построены
5. После заполнения заглушек, нам нужна Апликация, которая будет работать с этими 5ю rout
App.js -> class -> render -> switch -> Rout - его последовательность не важна, важно в coursesNav соблюдать порядок
6.Вот так мы построили Rout из нашей схемы. Подключаем Bootstrap в Index.css файле, для отображения меню.
7.В директории Utilit строим input_element нужны параметры: name, value, type, handler, lable, Select и Options для написания файла.
